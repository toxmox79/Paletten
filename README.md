# Palettenerfassung Online

Eine Online-App zur Erfassung von Paletten mit Speditionen und Erfassern, geschützt durch Passwort.

## Installation

1. Klone das Repository.
2. Installiere die Abhängigkeiten: `npm install`
3. Starte den Server: `npm start`
4. Öffne http://localhost:3000 (du wirst zur Login-Seite weitergeleitet)

## Deployment auf OnRender

1. Pushe den Code auf GitHub.
2. Verbinde das Repo mit OnRender.
3. Setze den Build Command auf `npm install` und Start Command auf `npm start`.
4. Deploye.

Die App ist passwortgeschützt, und Daten werden in SQLite gespeichert.
